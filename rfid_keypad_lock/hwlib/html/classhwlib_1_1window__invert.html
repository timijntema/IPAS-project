Detailed Description

window_invert (invert writes to a window)

A window_inverts writes inverted to its larger window.

Constructor & Destructor Documentation

hwlib::window_invert::window_invert ( window &  w)
inline

create a window_invert from a window

This call constructs a window_invert from a window. The foreground and background color are copied from the larger window, but writes to the window_invert write the invert pixel to the underlying window.
