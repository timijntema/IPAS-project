Detailed Description

hwlib implementation for the Arduino Due

This namespace contains the hwlib implementation of the pins, timing

and (sotware) UART output for the Arduino Due (ATSAM3X8E chip). The first wait call configures the chip to run at 12 MHz, from its internal (calibrated) RC oscillator.

The port and pin parameters to the constructors of the pin classes can either use to the ATSAM3X8E ports and pins, or the Arduino names.

The Due has an on-board orange LED connected to port 1 pin 27.

The chip runs at 3.3 Volt and that is the level on its IO pins.

The ATSAM3X8E chip has a watchdog system that is enabled by default. If left alone, the watchdog will reset the chip after a short time. To prevent this, all Due applications start with these lines:

// kill the watchdog
WDT->WDT_MR = WDT_MR_WDDIS;

Arduino Due pin names.

These are the pins of an Arduino Due board. Digital pins d0..d13, analog input pins A0..A5, SCL, SDA, TX (=D1), RX (=D0), LED (=D13), SCK (=D13), MISO (=D12), MOSI (=D11), SS (=D10).
